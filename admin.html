Getting a GitHub access token

1. Go to github.com/settings/tokens?type=beta (Fine-grained tokens)
2. Click Generate new token
3. Under "Repository access", choose Only select repositories → pick your site's repo
4. Under "Permissions" → Repository permissions → set Contents to Read and write
5. Click Generate token and copy it — paste it into the Access token field here

Keep this token private. Don't share this admin page's link publicly, and don't paste the token anywhere else.
